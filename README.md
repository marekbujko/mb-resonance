# Online Radio - Resonance
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/license/mit/)
[![GitHub stars](https://img.shields.io/github/stars/marekbujko/mb-resonance?style=social)](https://github.com/marekbujko/mb-resonance/stargazers)

Online Radio code based on the Web UI from RGB Binary Clock project.

It supports responsive design for iPhone and also works as a web app when shared on iOS and macOS.

Support for albums from Archive.org links has also been added, including favorites support (button to switch is at the bottom right).
